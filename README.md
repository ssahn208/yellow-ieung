# Yellow Ieung (ㅇ)

Interactive generative art exploring Korean letter 'ㅇ' (ieung) as glowing yellow stars in cosmic space.

## 🎨 작품 설명

28개의 노란 원(ㅇ)이 파란 우주 공간에서 서로 다른 속도와 방향으로 회전하며, 중심을 지날 때마다 D minor 음계의 부드러운 피아노 음을 생성합니다.

### 핵심 컨셉: ㅇ (이응)의 삼중 의미

**1. 음성학적 (Phonetic)**
- **초성**: 침묵 (mute) - 소리 없는 자음
- **중/종성**: 'NG' 음 - 울림소리 (resonance)
- → **침묵에서 소리로의 전환**을 상징

**2. 시각적 (Visual)**
- **원형**: 한글에서 유일한 완전한 원
- **Yellow**: 빛나는 노란 별들 - 어둠 속의 빛
- **Glow**: 살아 숨쉬는 빛의 펄스
- → **빛과 소리의 시각화**

**3. 우주론적 (Cosmic)**
- **동양 철학**: 원은 우주(宇宙)의 상징
- **28수 (二十八宿)**: 동양 천문학의 28개 별자리
- **훈민정음 28자**: 자음 17자 + 모음 11자 (1443년 창제 당시)
- **무한**: 시작도 끝도 없는 순환
- → **우주의 영원한 리듬**

### 색상의 의미

**Yellow (노란색):**
- 빛, 에너지, 생명력의 상징
- 어둠(침묵) 속에서 빛나는 소리
- 우주의 별 - 멀리서 보이는 노란 빛
- 희망과 각성

**Grayish Blue (회청색 배경):**
- 우주의 깊이와 무한함
- 명상적이고 고요한 공간
- 노란 이응들을 돋보이게 하는 캔버스

### 에릭 사티의 미니멀리즘

이 작품은 Erik Satie의 "Gymnopédies"에서 영감을 받았습니다:
- 단순한 형태(원)에서 생성되는 풍부한 음향
- 반복과 변주를 통한 명상적 경험
- 느린 템포와 공간감
- "침묵도 음악이다" - 이응의 mute에서 'NG' 소리로

## 🎹 기술 스택

### 비주얼
- **p5.js 1.9.4**: Canvas 기반 2D 렌더링 엔진
- **HTML5 Canvas API**: 커스텀 글로우 효과 (shadowBlur, shadowColor)
- **삼각함수**: 타원 궤도 시뮬레이션 (천체 물리학 기반)
- **한글 타이포그래피**: 'ㅇ' 형태를 기반으로 한 비주얼 언어
- **Color Palette**: 
  - Yellow (#ffff00) - 이응 원들
  - Grayish Blue (rgba(84,119,146,25)) - 우주 배경

### 오디오
- **Web Audio API**: 
  - OscillatorNode (사인파 생성기)
  - GainNode (ADSR 엔벨로프)
  - BiquadFilterNode (로우패스 필터, 1200Hz)
- **D minor modal scale**: 12음 기반 미니멀 멜로디 시스템
- **확률적 화성**: 3도(40%), 5도(25%) 자동 생성

### 인터랙션
- **반응형 Canvas**: 브라우저 리사이즈 자동 대응
- **클릭 이벤트**: 침묵(mute)에서 소리로 전환

## 🔬 기술적 세부사항

### 애니메이션 구조
- **28개 독립 궤도**: 각 노란 원(ㅇ)은 고유한 회전 방향과 속도
  - 28수 (二十八宿): 동양 천문학의 28개 별자리
  - 훈민정음 28자: 자음 17 + 모음 11 (1443년 창제)
  - 홀수 번째: 시계방향 (dir = 1)
  - 짝수 번째: 반시계방향 (dir = -1)
- **타원 궤도**: `rx = 250 + i * 20`, `ry = 120 + i * 10`
  - 태양계 행성 궤도에서 영감
- **회전 변환**: 각 궤도마다 `tilt = i * PI / 28` 적용
- **프레임레이트**: 60 FPS (부드러운 천체 운동)

### 사운드 생성 로직
- **트리거 조건**: 노란 원이 화면 중심 Y축 통과 시 (±8px)
  - 우주의 중심을 지나갈 때 소리 발생
- **음계**: D3~A5 범위의 12음 (D minor/modal)
  - D minor: 명상적이고 사색적인 음색
- **화성**: 
  - 3도 추가 (40% 확률) - 부드러운 울림
  - 5도 추가 (25% 확률) - 공간감
- **엔벨로프**: 
  - Attack: 0.02초 (피아노 타격 시뮬레이션)
  - Decay: 전체 duration의 30%
  - Release: 지수 감쇠 (자연스러운 소멸)
- **필터**: 1200Hz 로우패스 (따뜻한 피아노 음색)

### 시각 효과
- **글로우 펄스**: `sin(frameCount * 0.02 + i * 0.7)` 
  - 별빛의 깜박임 효과
- **크기 변조**: baseSize ± 30%
  - 호흡하는 듯한 유기적 움직임
- **노란색 빛**: Yellow (#ffff00) with dynamic alpha
  - 빛나는 별 같은 느낌
- **알파 블렌딩**: 배경 `rgba(84, 119, 146, 25)`
  - 궤적이 서서히 사라지는 효과 (우주의 여운)

## 🌌 예술적 의도

### 1. 빛과 소리의 변증법

```
어둠/침묵 (파란 우주)
    ↓
  클릭 (사용자 개입)
    ↓
빛/소리 (노란 이응)
    ↓
  순환 (원의 회전)
    ↓
우주적 화음
```

### 2. 한글의 과학성 × 사티의 직관성

- **한글**: 조음 위치를 시각화한 과학적 문자
- **이응 (ㅇ)**: 목구멍에서 나는 소리를 원으로 표현
- **노란색**: 빛 = 소리의 시각적 은유
- **사티**: 직관적이고 반규칙적인 작곡
- → **빛과 소리, 과학과 예술의 만남**

### 3. 우주의 노란 별들

| 동양 (이응 ㅇ) | 서양 (사티) | 작품 (Yellow Ieung) |
|--------------|------------|-------------------|
| 28수 (二十八宿) | 영원 회귀 | 28개 궤도 순환 |
| 훈민정음 28자 (17+11) | 미니멀리즘 | 어둠 속의 빛 |
| 천원지방 | Gymnopédies | 우주의 노란 별들 |
| 빛 = 소리 | 침묵 = 음악 | 침묵→소리 전환 |

### 4. 28이라는 숫자

**동양 천문학:**
- **28수 (二十八宿)**: 달의 공전 궤도를 28개 구역으로 나눈 별자리
- 중국, 한국, 일본의 전통 천문학 체계
- 4방위 × 7수 = 28개 별자리

**훈민정음 (1443):**
- **28자**: 자음 17자 + 모음 11자
- 자음 17: ㄱㄴㄷㄹㅁㅂㅅㅇㅈㅊㅋㅌㅍㅎ ㆁㆆㅿ
- 모음 11: ㅏㅑㅓㅕㅗㅛㅜㅠㅡㅣ ㆍ(아래아)
- 세종대왕이 창제한 완전한 문자 체계

**완전수 (Perfect Number):**
- **정의**: 자기 자신을 제외한 약수의 합이 자신과 같은 수
- **28의 약수**: 1, 2, 4, 7, 14, 28
- **계산**: 1 + 2 + 4 + 7 + 14 = 28 ✓
- **희귀성**: 28은 두 번째 완전수 (첫 번째는 6)
- **의미**: 고대 그리스 피타고라스 학파에서 우주의 조화와 완전함을 상징

## 🎯 작품이 제기하는 질문들

1. **빛은 소리를 보는 것인가?**
   - 노란 이응 = 시각화된 소리
   - 글로우 펄스 = 소리의 파동

2. **침묵 속에 빛이 있는가?**
   - 이응(ㅇ)은 초성에서 mute
   - 하지만 노란색으로 빛남
   - 침묵도 하나의 표현

3. **우주의 별들은 무엇을 노래하는가?**
   - 28개 별 = 28수 = 28자모
   - 각자의 궤도 = 각자의 멜로디
   - 우주적 화음

4. **28은 왜 완전한가?**
   - 28수: 동양 천문학의 완전한 체계
   - 훈민정음 28자: 한글의 원래 완전한 체계
   - 완전수: 1+2+4+7+14=28 (약수의 합)
   - 우주의 조화와 균형을 수학적으로 표현
